# mink

A port of [pink](https://github.com/stephane-caron/pink) for MuJoCo.

## Installation

```bash
pip install -e ".[examples]"
```

## Examples

```python
mjpython examples/arm_ur5e.py  # On macOS
python examples/arm_ur5e.py  # On Linux
```
